# Казан · Бал на Лисій горі

Офлайн-довідник рецептів для настільної гри «Бал на Лисій горі». Відьми наосліп
тягнуть інгредієнти в казан → гравець вводить їх у застосунок → миттєво отримує
рецепт (назву, художній текст, ефект).

Жодного сервера, реєстрації чи інтернету — усі дані (8567 рецептів + 13 іконок)
бандляться в застосунок.

## Стек
- React Native + Expo + TypeScript (один код → iOS, Android, web)
- Повністю офлайн, стан — локальний React state
- Мова інтерфейсу — українська

## Запуск

```bash
npm install
npm start          # Expo Dev Server (QR для Expo Go)
npm run android    # Android
npm run ios        # iOS
npm run web        # браузер
```

Web-збірка:

```bash
npx expo export --platform web   # → dist/
```

## Як це працює

1. Зверху — вибір числа відьом N (2–5). Впливає лише на максимум інгредієнтів
   у режимі «Ковен».
2. Перемикач режиму: **Одиночна варка** (Solo) / **Ковен** (Coven).
3. Сітка з 13 іконок інгредієнтів. Тап — додати в казан (бейдж із кількістю),
   довгий тап або «−» на чипі казана — прибрати один.
4. **Зварити** активна лише за валідного набору:
   - Solo: 1–3 інгредієнти
   - Coven: 2–N інгредієнтів
5. Картка результату: назва, бейджі (Том, легендарний), художній текст (курсив),
   ефект за режимом. **Нова варка** очищає казан.

## Пошук рецепта

`id інгредієнтів → сортування → склейка через '+' → Map.get(key)` — O(1).
Повтори інгредієнтів зберігаються як повторювані id у ключі (мультимножина).

## Структура

```
App.tsx                      кореневий екран і стан
recipes.json                 8567 рецептів (бандлиться)
assets/ingredients/{id}.png  13 іконок (стиснуті до 512px)
src/
  data/
    types.ts                 типи
    ingredients.ts           майстер-список 13 інгредієнтів + require іконок
    recipes.ts               імпорт JSON, індекс Map, findRecipe()
    validation.ts            правила кількості за режимом
  components/
    TopControls.tsx          вибір N + режим
    IngredientGrid.tsx       сітка іконок із бейджами
    Cauldron.tsx             поточний набір + лічильник
    ResultCard.tsx           картка рецепта
  theme.ts                   темна folk-horror тема
```
